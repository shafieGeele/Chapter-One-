Name: Section ‚Üí Wrapper / Layout Component

Purpose:

Wraps a section of the page

Adds a title (h2)

Renders any nested content (children)

Forwards extra props (id, className, etc.) using ...attr

Pattern:

Reusable wrapper

Props forwarding (...attr)

Children composition

Example Usage:



function Section ({title, children, ...props}){
    return <section {...props}>
        <h2>{title}</h2>
        {children}
    </section>
}

export default Section
